An R package with custom ggplot2 themes.

### Usage
#### Installation  

Install dependencies:

```
pak::pkg_install(c("ggplot2","scales","extrafont"))
```

This package can be installed as follows:

```
install.packages("pak")
pkg::pkg_install("royfrancis/plummy")
```
